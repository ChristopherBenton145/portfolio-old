## Getting Started

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/ChristopherBenton145/portfolio
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the app
   ```js
   npm start
   ```
